## Overview
Bouncing Ball is a delay plugin that uses a physics simulation of a ball bouncing around a room to generate a set of delay times.
By altering the parameters of the simulation (gravity, wall damping, ball speed & angle), a variety of interesting rhythmic effects can be achieved.

Another key feature of Bouncing Ball is that the output of its filter for each delay is fed to the subsequent delay, meaning sounds can develop into resonant ambient textures.

## Installation
### VST3
 - Download 'BouncingBall.vst3.zip' from the most recent release in the 'Releases' tab
 - Unzip the file and place it in your VST3 plugin folder, the location of which should be:
<p align="center">
  C:\Program Files\Common Files\VST3
</p>
<p align="center">
  (If this folder doesn't exist you can manually create it)
</p>

- Scan for the plugin in your DAW

### AU (for MacOS)
 - Download 'BouncingBall.component.zip' from the most recent release in the 'Releases' tab
 - Unzip the file and place it in your AU plugin folder, the location of which should be:
<p align="center">
  /Library/Audio/Plug-Ins/Components
</p>

- Scan for the plugin in your DAW
